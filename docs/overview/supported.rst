Architectures and Operating Systems
***********************************

Architectures
-----------------------

* - The FD.io VPP platform supports:

   * - x86/64
   * - ARM
     
Operating Systems and Packaging
-------------------------------

FD.io VPP supports package installation on the following 
recent LTS operating systems releases:

* - Operating Systems:

   * - Debian
   * - Ubuntu 
   * - CentOS 
   * - OpenSUSE
